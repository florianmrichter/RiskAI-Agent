# Bulk-Validator Skill

Portfolio-weiter Health-Check für alle FMEA-Projekte.

## Trigger-Words

- "Portfolio-Check"
- "alle Analysen prüfen"
- "Bulk-Validierung"
- "Health-Check"
- "alle Projekte validieren"

## Was wird geprüft?

1. Anlagendaten vorhanden und valide
2. FMEA-Daten in der Datenbank
3. Fehlermodi-Abdeckung pro Komponente
4. RPZ-Verteilung (plausibel?)
5. Maßnahmen für Hoch-Risiko-FMs
6. Report-Generierung

## Ausgabe

Tabellarische Übersicht aller Projekte mit Status (OK/WARN/FAIL) und Handlungsempfehlungen.
