# Domain-Wissen aus Kontext-Recherchen

Dieser Ordner enthält zentral gespeichertes Wissen aus Recherchen vor und während Risikoanalysen.

**Zweck:** Domain-spezifisches Wissen (Prozess, typische Gefahren, Regulierung, Fachbegriffe) für spätere Analysen verfügbar halten. Anlagendaten bleiben immer primäre Quelle – dieses Wissen dient als Ergänzung.

**Struktur:** Pro Domäne oder Prozesstyp eine Datei, z.B.:
- `verfahrenstechnik_esterifizierung.md` – Fischer-Veresterung, Ethylacetat
- `pharma_gmp.md` – (bei Bedarf)
- `verfahrenstechnik_destillation.md` – (bei Bedarf)

**Hinweis:** Der Agent ergänzt diese Dateien bei Recherchen. Keine manuellen Änderungen nötig.
