# Archiv: batch_fmea_minimal

**Datum:** 2026-03-02

Dieses Skript wurde versehentlich eingesetzt, um eine abgespeckte Risikoanalyse (1 Funktion, 1 Fehlermodus pro Komponente) durchzulaufen. Der Nutzer hat **ausschließlich vollständige** Risikoanalysen gewünscht (alle Funktionen, alle Fehlermodi mit Begründungen, Maßnahmen etc.).

**Nicht verwenden.** Die Risikoanalyse ist ausschließlich mit vollständiger FMEA pro Komponente durchzuführen (Agent analysiert jede Komponente, schreibt in `fmea_explicit.py`, Einspielung mit `insert_fmea_explicit`).
